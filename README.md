## Description

This project was bootstrapped with [Nest](https://github.com/nestjs/nest).

## Project structure
The project follows the standard structure based on a feature (resource) separation. There are root App resource and added Users resource.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
